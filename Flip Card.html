CFGS ASIR

El ciclo de Administración de Sistemas Informáticos en Red (ASIR) forma profesionales en la gestión y mantenimiento de redes, servidores y sistemas operativos. Incluye conocimientos en virtualización, seguridad informática, bases de datos, y servicios de red. Ideal para quienes buscan trabajar como técnicos en soporte, administradores de sistemas o consultores IT.

CFGS DAM

CFGS DAM

El ciclo de Desarrollo de Aplicaciones Multiplataforma (DAM) está orientado a la creación de software para distintos sistemas operativos (Windows, Linux, Android, etc.). Se estudian lenguajes como Java, Kotlin y SQL, además de desarrollo de interfaces, gestión de bases de datos y acceso a servicios en red. Es ideal para quienes quieren ser programadores o desarrolladores de apps.

CFGS DAW

CFGS DAW

El ciclo de Desarrollo de Aplicaciones Web (DAW) capacita para diseñar, programar y mantener sitios y aplicaciones web. Se trabajan tecnologías como HTML, CSS, JavaScript, PHP y MySQL, así como frameworks modernos. También se tratan aspectos de usabilidad, accesibilidad y posicionamiento web (SEO). Ideal para convertirse en desarrollador web full-stack.
